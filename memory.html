How to use Memory
  • 1. Select a Text: Choose a reading from the sidebar to load the deck.
  • 2. Targeted Practice: Read the text in the bottom pane. Click or tap individual words to select them, or drag your mouse (or finger on a screen) across the text to highlight a whole phrase. Then press Drill Selected Words.
  • 3. Flashcards: Press Spacebar (or tap/click the card) to reveal the translation.
  • 4. Grade Yourself: Press 1 if you remembered it. Press 2 if you forgot, and it will return to the queue.
